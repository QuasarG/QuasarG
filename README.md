## 精选项目

**[Taste Bank](https://github.com/QuasarG/taste-bank)** — 面向 coding agent 的开源前端风格库
把一套风格沉淀为结构化 pack（SKILL.md + design tokens + 模板快照），经 MCP server 直接递到 agent 手里。一次沉淀，处处复用。
`TypeScript` `Astro` `MCP`

**[rednote-copilot](https://github.com/QuasarG/rednote-copilot)** — 小红书广告博主的 AI Copilot
`Python` `JavaScript`

**[win-folder-manager](https://github.com/QuasarG/win-folder-manager)** — Windows 文件夹视觉自定义（fork）
`Python` `HTML`
